## Build
Brug følgende Gradle-kommando til at bygge MinePay:

```gradle
gradle build
```
> [!NOTE]  
> Den genererede MinePay-fil gemmes i mappen `build/libs/MinePay-Bukkit-{version}.jar`.
>
